Welcome to the Simple C++ Web Server

This is a lightweight HTTP server built using C++ that runs on localhost:8080.

About the Application

This application is a basic web server that:

  • Serves static files (like this HTML page) from the public directory for GET requests.
  • Handles POST requests by echoing back the data sent to it.
  • Reads the content of PHP files
  • Supports basic HTTP status codes:
    • 200 OK: Successful request
    • 404 Not Found: File not found
    • 405 Method Not Allowed: Unsupported HTTP method

How to use this server:

  1. Compile the server with make
  2. Run the server with ./liveserver
  3. Open your browser and visit http://localhost:8080
  4. To stop the server, press Ctrl+C in the terminal then make fclean
Note: This server is designed to work exclusively on Linux operating systems.
